# Ciclo

Plataforma social para acompanhamento e competição de estudos para concursos.

## Objetivo

Permitir que amigos registrem sessões de estudo, acompanhem frequência, tempo, questões, acertos e disputem rankings semanais e gerais.

## Recursos planejados

- Registro de sessões de estudo
- Controle de tempo estudado
- Registro de questões e acertos
- Ranking semanal
- Ranking geral
- Grupos privados
- Sequências de estudo
- Histórico por disciplina
- Estatísticas de desempenho

## Situação do projeto

Protótipo inicial em desenvolvimento.
